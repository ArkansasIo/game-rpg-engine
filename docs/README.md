# Eldiron Documentation

This folder contains the documentation website for Eldiron (Docusaurus).

## Local Development

```bash
npm install
npm run start
```

## Build Static Docs

```bash
npm run build
npm run serve
```

## Docs Structure

- `docs/`: product and usage documentation pages
- `blog/`: release notes and dev logs
- `src/`: site pages and React components
- `static/`: static images and assets
